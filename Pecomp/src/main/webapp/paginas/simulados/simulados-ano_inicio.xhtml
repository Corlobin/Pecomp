A flag de arquivamento indica se um arquivo deve ser apagado pelo sistema.

Atributos de momento são utilizados para indicar arquivos temporários.

Flags são bits ou campos pequenos que controlam alguma característica específica.

O atributo "tamanho do registro" indica o número de bytes no campo chave.

Sistemas operacionais modernos implementam todos os atributos possíveis a um arquivo.
